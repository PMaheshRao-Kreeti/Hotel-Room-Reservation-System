## recommended to use google chrome or firefox

* Yarn Version -> 1.22.19

* database used -> postgresql

* elastic search version -> 7.4.0

## setup instruction:

```bundle && yarn```

**make sure to start elasticsearch server before seeding the database**

```rails db:create```

```rails db:migrate```

```rails db:seed```
